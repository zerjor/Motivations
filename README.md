# Motivations
Команда                Описание
/start                 Приветственное сообщение
/help                  Справка
/motivation            Получить мотивирующую цитату


Для работы бота используются следующие библиотеки:

pyTelegramBotAPI (импортируется как telebot) – основная библиотека для взаимодействия с Telegram Bot API.

random – для случайного выбора цитаты.

config – модуль, содержащий токен бота(не библиотека, оттельный файл)
